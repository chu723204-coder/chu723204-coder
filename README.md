## 📌 프로젝트

### DevLink - 개발자 취준생 커뮤니티 플랫폼
> 1인 풀스택 개발 | Spring Boot · React · WebSocket · SSE · JWT · OAuth2 · PostgreSQL

- JWT 듀얼토큰 인증 구조 설계 (Access Token 메모리 저장 / Refresh Token HttpOnly Cookie)
- WebSocket STOMP 실시간 채팅 + ChannelInterceptor JWT 검증
- SSE 실시간 알림 (댓글/좋아요/스터디 지원/수락/거절)
- 카카오/네이버 OAuth2 소셜 로그인

[Backend](https://github.com/chu723204-coder/DevLink_Back) | [Frontend](https://github.com/chu723204-coder/DevLink_Front)
